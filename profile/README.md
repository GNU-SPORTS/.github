# ⚽ GNU-SPORTS ⚽

저희는 학교 체육시설의 불편한 예약, 사용 방식을 개선하기 위해서   

직접 학교 내 스포츠 시설 예약 및 커뮤티니 시스템을 직접 구현 하였습니다.
